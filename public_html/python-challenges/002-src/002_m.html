We are using the sieve of Eratosthenes algorithm to find the largest prime number under 8392842.

The sequence of images provides instructions on how to perform the algorithm.


Following the first image, cross out all the multiples of 2 in the list, except 2.


Following the second image, cross out all the multiples of 3, except 3.


Following the third image, cross out all the multiples of 5 (because we don't have 4 already), except 5.


Following the fourth image, we don't have any multiples of 7. Therefore, the list is "sublime", and all the remainings are prime numbers under 40.


Therefore, to find the largest prime number under 8392842, we just need to repeat the above steps.

However, this time the list starts from 2, ends at 8392842.


Here is the code solution: 

def factoriser(n):

    v = [True for i in range(n+1)]

    for i in range(2,n):
        if v[i]:
            k = 2
            while i*k <=n:
                v[i*k] = 0
                k+=1

    for i in range(n,2,-1):
        if v[i]:
            result = i
            break

    return result

print(factoriser(8392842)) #8392831

The answer is 8392831!
